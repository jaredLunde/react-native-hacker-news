Hacker News

A Hacker News app written in react-native and Dash

Getting started

 
# Clone the repo
git clone https://github.com/jaredLunde/react-native-hacker-news
# Enter the new directory
cd react-native-hacker-news
# Install and start an iOS app
yarn && yarn ios
